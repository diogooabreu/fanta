# Fanta – Product Page

## Purpose

This project is a product page developed as part of the "Rei dos Sites" course, with additional improvements implemented independently.

The main goals of this project are:

- practice semantic HTML structure
- reinforce CSS (without frameworks)
- train basic JavaScript DOM manipulation
- implement responsiveness
- expand the original guided project with custom improvements
- apply Git and GitHub workflow during development

This project is being progressively enhanced beyond the original course version.

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM manipulation)
- GSAP (animations)
- Git
- GitHub
